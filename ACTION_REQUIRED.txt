## 🎯 IMMEDIATE ACTION (Choose ONE)

### Option 1️⃣: MongoDB Atlas (Cloud - NO INSTALLATION)
**⏱️ Time: 5-10 minutes | 🎯 Easiest**

✅ **RECOMMENDED FOR QUICK START**

1. Go to: https://account.mongodb.com/account/register
2. Create free account
3. Create cluster (wait 2 min)
4. Copy connection string
5. Edit `server/.env` and replace:
   ```
   MONGO_URI=mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/digitalkhalane?w=majority
   ```
6. Run: `npm run dev` in server folder
7. Done! ✅

---

### Option 2️⃣: Docker (If you have Docker installed)
**⏱️ Time: 3 minutes**

```bash
cd d:\DigitalKhalane
docker-compose up -d mongodb
cd server
npm run dev
```

Keep `.env` as is (localhost:27017)

---

### Option 3️⃣: Install MongoDB Locally
**⏱️ Time: 10 minutes**

Windows:
1. Download: https://www.mongodb.com/try/download/community
2. Run installer
3. Accept defaults
4. MongoDB starts automatically

Keep `.env` as is (localhost:27017)
