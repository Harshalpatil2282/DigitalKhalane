# рҹҡҖ Get MongoDB Running - 3 Quick Options (Pick ONE)

Your DigitalKhalane server needs MongoDB. Choose the fastest option for you:

---

## вҡЎ Option A: MongoDB Atlas (Easiest - 3 minutes)

**No installation needed! Use free cloud MongoDB**

### Quick Setup
1. Go to: https://account.mongodb.com/account/register
2. Sign up (free)
3. Create cluster (click the big "CREATE" button)
4. Wait ~2 minutes for cluster to build
5. Click "Connect" вҶ’ Choose "Drivers" вҶ’ Copy connection string

### Update Your .env
Replace `MONGO_URI` in `server/.env`:
```
MONGO_URI=mongodb+srv://username:password@cluster0.xxxxx.mongodb.net/digitalkhalane?w=majority
```

вҡ пёҸ **IMPORTANT:** Replace `username` and `password` with your MongoDB Atlas credentials, and replace the cluster part with your actual cluster URL.

### Start Server
```bash
cd server
npm run dev
```

вң… **Done!** Server should start with `вң… MongoDB Connected Successfully`

---

## рҹҗі Option B: Docker (3 minutes if Docker installed)

**Easy one-command MongoDB startup**

### Prerequisites
- Docker installed? Check: `docker --version`
- If not installed: https://www.docker.com/products/docker-desktop

### Run MongoDB in Docker
```bash
# Navigate to project root
cd d:\DigitalKhalane

# Run MongoDB container
docker-compose up -d mongodb

# Verify it's running
docker ps
```

### Verify Connection
```bash
# This should work
mongosh "mongodb://localhost:27017/digitalkhalane"

# Type: exit
```

### Start Server
```bash
cd server
npm run dev
```

вң… **Done!** Server should start

**To stop MongoDB later:**
```bash
docker-compose down
```

## рҹ’» Option C: Installed MongoDB (10 minutes)

**If you have MongoDB installed on your computer**

### Windows
```bash
# Check if MongoDB is running
# Open Services app вҶ’ Search "MongoDB" вҶ’ Look for "MongoDB Server"

# If not running, start it:
# Right-click "MongoDB Server" вҶ’ Start

# Verify

# Should show: test>
```

### Mac (Homebrew)
```bash
# Check if running
brew services list | grep mongodb

# Start if not running
brew services start mongodb-community

# Verify
mongosh
```

### Linux
```bash
# Check if running
sudo systemctl status mongodb

# Start if not running
sudo systemctl start mongodb

# Verify
mongosh
```

### Start Server
```bash
cd server
npm run dev
```

вң… **Done!** Server should start

## вң… Verify It's Working

After starting MongoDB (using ANY option above), run:

```bash
# Test: Should return 200 OK with token
curl -X POST http://localhost:5000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"admin@digitalkhalane.in","password":"khalane@2024"}'
```

**Response should be:**
```json
{
  "success": true,
  "message": "Login successful",
  "token": "eyJhbGc...",
  "user": {...}
}
```

If you see вң… **200 OK** - SUCCESS! Database is connected!
